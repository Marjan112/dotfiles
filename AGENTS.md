# AGENTS.md

## Commit messages

- The commit messages should always be in a format like this: [DIRNAME] message.
- The DIRNAME is the name of directory where a change, deletion or addition happened.
- If files from multiple directories were changed, simply just generate a commit message without the [DIRNAME] prefix.
- If a file was changed outside of the directories in this repository, simply just generate a commit message without the [DIRNAME] prefix.
